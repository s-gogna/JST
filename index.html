What is JST?

JST is a compiler for ANSI C written in Python using the PLY library. It is a group project for the Compiler Construction course taught by Dr. Fred Harris at the University of Nevada, Reno.

Requirements

  • Python3.4+ sudo apt-get install python3
  • PLY pip3 install ply
  • bintrees pip3 install bintrees

Features

  • Variable declaration int x;
  • Pointers float *x;
  • Function declaration and implementation int isExciting() { return 1; }
  • Calling functions isWorking();

Assumptions

  • Programs are reasonably close to C syntax (as the grammar allows some very strange programs to be parsed)

Restrictions

  • Parse terminates if an error is raised
  • Typedefs not implemented
  • Structs and unions not implemented

Implementation

  • Scanner
  • Parser
  • Symbol
  • Scope
  • Symbol Table

To-Do List
