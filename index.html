What is JST?

JST is a compiler for C written in Python3 using the PLY library.

Requirements

  • Python3.4+
  • PLY pip3 install ply
  • bintrees pip3 install bintrees

Features

  • Variable declaration int x;
  • Pointers float *x;
  • Function declaration and implementation int isExciting() { return 1; }

To-Do List

  • Typedef support
  • Calling functions
  • Function pointers
  • Structs and unions

Preferred Method of Evaluation

We recommend using PyCharm to evaluate or contribute to the compiler. PyCharm's built-in functionality for testing also helps with making sure new code works.

Testing

To run all unit tests, switch to the top of project directory and run:

python3 -m unittest

To test just the Symbol Table using it's custom test driver, run:

python3 bin/symbol_table_driver.py
